# CodeMorph AI

AI-powered code translator and analyzer built with React, Node.js, Supabase, and Google Gemini.

🔗 **Live Demo:** [https://code-morph-ai-89a1.vercel.app](https://code-morph-ai-89a1.vercel.app)

## Features

- Translate code between 22 programming languages
- Explain code in plain language
- Analyze time/space complexity and code quality
- Detect bugs, security issues, and performance problems
- Translation history with search

## Tech Stack

- **Frontend:** React + Vite → Vercel
- **Backend:** Node.js + Express → Render
- **Database:** Supabase PostgreSQL
- **AI:** Google Gemini 2.5 Flash
